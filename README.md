# TahoeJesse
TahoeJesse is a client portal and task management app for a property management service in the greater Lake Tahoe area.
Clients can log in with google Oauth
### [Deployed Site](https://tahoe-jesse.fly.dev)

## Technologies Used
- HTML
- CSS
- JavaScript
- EJS
- Express
- Node
- MongoDB
- Mongoose
- Google Oauth
